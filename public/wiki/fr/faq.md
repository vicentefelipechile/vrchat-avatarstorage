# Questions fréquemment posées

### Pourquoi ne sont-ils pas inclus dans le téléchargement ?
Poiyomi et VRCFury sont des outils tiers constamment mis à jour. Les créateurs d'avatars ne peuvent pas légalement les redistribuer, vous devez donc les télécharger directement depuis leurs sources officielles.

### Que contient un téléchargement d'avatar typique ?
Un téléchargement d'avatar contient généralement : fichiers FBX/Blend du modèle 3D, textures, matériaux, prefabs Unity, et parfois des fichiers de configuration. Il ne contient PAS de shaders ou plugins comme Poiyomi ou VRCFury.

### Que se passe-t-il si je n'installe pas Poiyomi ?
Si un avatar utilise Poiyomi et que vous ne l'avez pas installé, vous verrez des matériaux en magenta (rose vif) dans Unity, indiquant que le shader est manquant. L'avatar ne s'affichera pas correctement.

### Ai-je besoin de la version Pro de Poiyomi ?
Cela dépend de l'avatar. La plupart fonctionnent avec la version gratuite. Si un avatar nécessite la version Pro, le créateur l'indiquera dans la description. Les fonctionnalités Pro incluent les effets de réfraction, le shader de fourrure et les effets géométriques.

### Que sont les fichiers .spp (Substance Painter) ou Photoshop (.psd) ?
Le fichier **.spp** est le projet de texture pour le programme "Substance 3D Painter" (tout comme un **.psd** l'est pour Photoshop). Ces fichiers **NE SONT PAS nécessaires** pour uploader l'avatar dans VRChat. Ils sont inclus uniquement si vous souhaitez éditer, repeindre ou créer des textures personnalisées pour les vêtements, le corps ou les cheveux. Si vous ne savez pas comment modifier les textures, vous pouvez complètement les ignorer.

### Qu'est-ce que le fichier .blend ?
Le fichier **.blend** est le modèle 3D original sauvegardé au format Blender (un logiciel de modélisation 3D gratuit). Tout comme les fichiers de texture, **vous N'EN AVEZ PAS BESOIN pour installer ou uploader votre avatar dans VRChat**. Il est fourni au cas où vous voudriez ouvrir le modèle dans Blender pour modifier le maillage (mesh), créer des vêtements sur mesure ou modifier les os.

> [!NOTE]
> Note
> Pour uploader un avatar dans VRChat, **vous n'avez besoin que d'utiliser le package Unity (`.unitypackage`)**. Les outils d'édition comme Substance Painter (.spp) ou Blender (.blend) sont totalement optionnels et destinés aux créateurs souhaitant apporter des modifications avancées.
