# Perguntas Frequentes

### Por que não estão incluídos no download?
Poiyomi e VRCFury são ferramentas de terceiros que são constantemente atualizadas. Os criadores de avatares não podem redistribuí-las legalmente, então você deve baixá-las diretamente de suas fontes oficiais.

### O que um download típico de avatar inclui?
Um download de avatar geralmente inclui: arquivos FBX/Blend do modelo 3D, texturas, materiais, prefabs do Unity e, às vezes, arquivos de configuração. NÃO inclui shaders ou plugins como Poiyomi ou VRCFury.

### O que acontece se eu não instalar o Poiyomi?
Se um avatar usar Poiyomi e você não tiver instalado, verá materiais magenta (rosa brilhante) no Unity, indicando que o shader está faltando. O avatar não ficará com a aparência correta.

### Eu preciso da versão Pro do Poiyomi?
Depende do avatar. A maioria funciona com a versão gratuita. Se um avatar exigir a versão Pro, o criador indicará na descrição. Os recursos da versão Pro incluem efeitos de refração, fur shader e efeitos de geometria.

### O que é o arquivo .spp (Substance Painter) ou arquivos do Photoshop (.psd)?
O arquivo **.spp** é o projeto de textura do programa "Substance 3D Painter" (assim como um **.psd** é do Photoshop). Esses arquivos **NÃO são necessários** para enviar o avatar para o VRChat. Eles estão incluídos apenas caso você queira editar, repintar ou criar texturas personalizadas para as roupas, o corpo ou o cabelo. Se você não sabe editar texturas, pode ignorá-los completamente.

### O que é o arquivo .blend?
O arquivo **.blend** é o modelo 3D original salvo no formato do Blender (um software gratuito de modelagem 3D). Semelhante aos arquivos de textura, **você NÃO precisa dele para instalar ou carregar seu avatar no VRChat**. Ele é fornecido caso você deseje abrir o modelo no Blender para editar a malha (mesh), fazer roupas sob medida ou modificar os ossos.

> [!NOTE]
> Nota
> Para enviar um avatar para o VRChat, **você deve usar apenas o pacote do Unity (`.unitypackage`)**. Ferramentas de edição como Substance Painter (.spp) ou Blender (.blend) são totalmente opcionais e destinadas a criadores que desejam fazer modificações avançadas.
