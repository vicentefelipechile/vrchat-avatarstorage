# Unity Hub エラーの修正

Unity Hub が開かない、無限のロード画面で停止する、またはログインエラーでプログラムが使用できない場合、最も効果的な解決策は**クリーンインストール**を実行することです。

すべての一時ファイルと破損した設定を削除する方法は次のとおりです。

## 方法 1: クリーンインストール (すべての痕跡を削除)

Unity Hub が再び機能するように、次の手順に注意深く従ってください。

### 1. Unity Hub をアンインストールする
> [!WARNING]
> 警告
> この手順では、**公式の Windows アンインストーラー**（*設定 -> アプリ* または *コントロール パネル*から）を使用する必要があります。IObit Uninstaller や Revo Uninstaller などの**サードパーティ製プログラムは使用しないでください**。必要なレジストリ キーが削除され、問題が悪化する可能性があります。

- **Windows の設定** -> **アプリ** に移動します。
- リストから「Unity Hub」を見つけて、**[アンインストール]** をクリックします。

### 2. 残留ディレクトリを削除する
アンインストール後も、Unity はシステムに隠し設定 (キャッシュ) フォルダーを残します。これらを手動で見つけて削除する必要があります。

Windows のエクスプローラーを開き、次の各アドレスを上部のバーにコピーして、Enter キーを押します。**フォルダーが存在する場合は、完全に削除してください。**

- `C:\Program Files\Unity`
- `C:\Program Files\Unity Hub`
- `%USERPROFILE%\AppData\Roaming\Unity`
- `%USERPROFILE%\AppData\Roaming\Unity Hub`
- `%USERPROFILE%\AppData\Local\Unity`
- `%USERPROFILE%\AppData\Local\Unity Hub`

*(注：`%USERPROFILE%` パスは、Minecraftにmodをインストールする際に `%appdata%` を使用するのと同じように、エクスプローラーのアドレスバーに直接コピー＆ペーストすることができ、現在のユーザーフォルダーに自動的に移動します)。*

### 3. Unity Hub を再インストールする
システムから Unity ファイルが完全に消去されたら:
1. [Unity の公式 Web サイト](https://unity.com/download) に移動し、最新バージョンの Unity Hub をダウンロードします。
2. インストーラーを実行し、通常の手順に従います。
3. すべてが正しくインストールされるのを待ち、再度ログインしてエラーが修正されたことを確認します。
