# Esska Desktop Puppeteer

<span class="badge">ユーティリティ</span>

## これは何ですか？
**Esska Desktop Puppeteer**は、**Esska**によって作成されたVRChatのデスクトップユーザー向けの高度なツールです。これは2つの部分からなるシステム（デスクトップアプリケーションとアバターパッケージ）で構成されており、コンピューターのマウスを使用してアバターの特定の体の部分を制御できるようにします。これにより、通常は仮想現実（VR）ユーザーにのみ利用可能なレベルの精度と表現力が提供されます。

## 何のために使うのですか？
- **手足の制御：** マウスを使って直接、アバターの腕や手を独立して正確に動かすことができます。
- **カスタムパーツ：** 耳、尻尾、アクセサリーなど、追加のアバターパーツの制御を簡単にします。
- **デスクトップでのVRシミュレーション：** その主な目的は、デスクトップユーザーに、まるでVRでプレイしているかのように見える自由な動きを提供することです。
- **ヘッドトラッキング：** TrackIRデバイスのサポートを備えており、実際の動きに応じてアバターの頭部を動かすことができます。

> [!NOTE]
> メモ
> このツールは、**OSC（Open Sound Control）**を使用して、デスクトップアプリケーションからVRChatクライアントにパラメーターを送信します。VRChatのラジアルメニューでOSCオプションが有効になっていることを確認してください。

## 前提条件
開始する前に、次の要件を満たしていることを確認してください。
- **オペレーティングシステム：** Windows 10 または Windows 11。
- **ソフトウェア：** [Microsoft .NET 9.0 Desktop Runtime](https://dotnet.microsoft.com/download/dotnet/9.0)がPCにインストールされていること。
  - *ダウンロード方法：* リンクを開き、「**.NET Desktop Runtime**」というセクションを探します。その下の小さな表の「Windows」の行にある「**x64**」リンクをクリックしてインストーラーをダウンロードします。
- **ハードウェア：** センターボタン（スクロールホイール）付きのマウス。
- **VRChat SDK：** Unityプロジェクトにインストールされていること（VCC経由）。
- **アバター：** 互換性のあるヒューマノイドアバター（標準的な人間のプロポーションで最適に機能します）。

## ステップバイステップ・インストールガイド

インストールプロセスは、Unityでのアバターの準備とデスクトップアプリケーションのセットアップという2つの主要な部分に分かれています。

### パート1：アバターへのインストール（Unity）
1. **パッケージのインポート：** 公式ページから「Base Package」をダウンロードし、`.unitypackage`ファイルをUnityプロジェクトの`Assets`フォルダーにドラッグします。
2. **アバターに追加：** Esska Desktop Puppeteerパッケージに含まれているプレハブを見つけ、`Hierarchy`（階層）ツールでアバターにドラッグします。
3. **パラメータの設定：** システムはOSCパラメーターを使用します。アバターに新しいコントロールを収容するのに十分なパラメーターメモリ（Parameters Memory）があることを確認してください。
4. **アバターのアップロード：** プレハブが正しく配置され設定されたら、通常通りにアバターをVRChatにアップロードします。

### パート2：デスクトップアプリケーションのセットアップ
1. **アプリのダウンロード：** 「Esska Desktop Puppeteer App」アプリケーションをダウンロードします。
2. **実行：** VRChatセッションの前またはセッション中にPCでアプリケーションを開きます。
3. **VRChatでOSCを有効にする：** VRChat内でラジアルメニューを開き、`Options` -> `OSC`に移動して、**Enabled**に設定されていることを確認します。
4. **使い方：** アプリケーションの指示に従って、マウスのボタン（特に中央ボタン）とキーボードを使用して、アバターの手足の移動を開始します。

> [!WARNING]
> 警告：プライバシーとコントロール
> アプリケーションは、VRChatウィンドウがアクティブなときに動作できるように、キーボードとマウスの入力（グローバル フック）を「リッスン」する必要があります。作成者は個人データを収集しないと述べていますが、他のアプリケーションとの干渉を避けるためにプログラムの動作方法を知っておくことが重要です。
