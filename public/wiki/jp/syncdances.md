# SyncDances

<span class="badge">ツール</span>

## 概要

SyncDancesは、アバターが完璧に同期して踊ることができるVRChat用のUnityプレハブです。1人のプレイヤーがダンスを開始すると、システムをインストールしている全員が同時に踊り始めます。

> [!NOTE]
> SyncDancesは、[CuteDancer](https://github.com/Krysiek/CuteDancer)プレハブからインスピレーションを得て作成されました。

## 用途

- VRChatでの複数プレイヤー間の同期ダンス
- 1人が制御し、残りが追従する送信機・受信機システム
- ダンス速度の制御（同期済み）
- カスタムダンス用の24個のスロット

## 主な特徴

| 特徴                 | 説明                                                   |
| -------------------- | ------------------------------------------------------ |
| **同期**             | システムを持つすべてのプレイヤーが同時に踊ります       |
| **速度制御**         | ダンスを加速、減速、またはフリーズさせることができます |
| **カスタムスロット** | 独自のダンスを追加するための24個のスペース             |
| **Quest互換性**      | Questで動作します（ただし非推奨）                      |
| **複数のバージョン** | VRCFuryおよびModular Avatarに対応                      |

## 要件

- プロジェクトに**VRCFury**がインストールされていること（推奨）
- オプション：自動インストールのための**Modular Avatar**

## インストール方法

### VRCFuryを使用する方法（推奨）

1. パッケージから`SyncDancesPrefab PC (VRCFURY)`ファイルをダウンロードします
2. Unityでプレハブをアバターにドラッグ＆ドロップします
3. 完了です！アバターをアップロードする準備が整いました

> [!IMPORTANT]
> アイテムファイルを個別にインストールしないでください。メインのプレハブのみを使用してください。

### Modular Avatarバージョン

VRCFuryの代わりにModular Avatarを使用する場合：

- 以下のリンクから特定のバージョンを見つけてください：[SyncDances Modular Avatar](https://booth.pm/en/items/6311129)

## 使い方

1. アバターにプレハブをインストールします
2. VRChatメニューを使用してダンスを選択します
3. あなたが「送信機」である場合、他の「受信機」は同期して踊ります

### 送信機・受信機システム

- **1人のプレイヤーがアンテナ（送信機）として機能** - どのダンスを再生するかを制御します
- **他のプレイヤーは受信機** - 信号を受信して同期して踊ります

> [!TIP]
> 送信範囲を広げるには、すべての送信機と受信機を結合してください。ただし注意してください！VRChatのバグによりクラッシュする可能性があります。

## 速度制御

バージョン4.0以降、SyncDancesには速度制御が含まれています：

- **0%**: フリーズ
- **100%**: 通常速度
- **100%以上**: 加速ダンス

> [!WARNING]
> 速度制御は、SyncDances 3.1以前を使用している人には機能しません。彼らはデフォルトの速度で踊ります。

## よくあるエラー

### プレイヤーが同期しない

- 全員が同じバージョンの SyncDances を使用しているか確認してください
- 送信機が範囲内にいるか確認してください
- 3.1を使用しているプレイヤーは速度を制御できません

### アバターが固まる

- バージョンの不整合が原因の可能性があります
- プレハブが正しくインストールされているか確認してください

### カスタムエモートが動作しない

- 正しいスロットを使用しているか確認してください
- 一部のエモートにはVRCFuryのインストールが必要です
