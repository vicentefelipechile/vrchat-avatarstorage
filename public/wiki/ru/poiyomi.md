# Poiyomi Toon Shader

<span class="badge badge-blue">ЗАВИСИМОСТЬ</span>

## Что это?
Poiyomi - это шейдер для Unity, специально разработанный для VRChat. Он позволяет создавать стилизованные и мультяшные образы на аватарах с продвинутыми визуальными эффектами.

## Для чего это?
- Настраиваемое стилизованное затенение (toon, реалистичное, плоское)
- Специальные эффекты: контуры, декали, блестки, искры
- Поддержка AudioLink (эффекты, реагирующие на звук)
- Физически точные отражения и блики
- Оптимизирован для производительности VRChat

> [!WARNING]
> ОЧЕНЬ ВАЖНО
> Poiyomi НЕ включен в файлы аватаров, которые вы загружаете. Вы должны установить его самостоятельно в Unity ПЕРЕД открытием аватара.

## Где получить?
- **Бесплатная версия:** [GitHub - Poiyomi Toon Shader](https://github.com/poiyomi/PoiyomiToonShader)
- **Pro версия:** [Patreon - Poiyomi](https://www.patreon.com/poiyomi)

## Как установить?
1. Скачайте файл .unitypackage с GitHub или Patreon
2. Откройте ваш проект Unity
3. Импортируйте пакет: Assets → Import Package → Custom Package
