# Modular Avatar

<span class="badge">ИНСТРУМЕНТ</span>

## Что это?

Modular Avatar — это набор **неразрушающих** инструментов для модуляризации ваших аватаров VRChat и распространения компонентов аватара. С Modular Avatar добавление новой одежды или фишки к вашему аватару так же просто, как перетаскивание.

> [!NOTE]
> Modular Avatar работает через систему **Non-Destructive Modular Framework (NDMF)**, которая обрабатывает аватар во время сборки без изменения исходных файлов.

## Для чего это нужно?

- Установка одежды и аксессуаров в один клик через **drag-and-drop**
- Организация аниматоров: разделите FX аниматор на несколько под-аниматоров и объедините их во время выполнения
- Автоматическая настройка меню VRChat
- Система **тоглов** для активации/деактивации объектов и блендшейпов
- Реактивные компоненты, реагирующие на изменения аватара
- Распространение префабов с автоматической установкой

## Основные функции

| Функция                     | Modular Avatar     | VRCFury          |
| --------------------------- | ------------------ | ---------------- |
| **Установка одежды**        | Да (drag-and-drop) | Да (в один клик) |
| **Система тоглов**          | Да (продвинутая)   | Да (базовая)     |
| **Организация аниматоров**  | Да (объединение)   | Нет              |
| **Автоматические меню**     | Да (полная)        | Да (базовая)     |
| **Неразрушающий процесс**   | Да (NDMF)          | Да               |
| **Блендшейп синхронизация** | Да                 | Нет              |
| **Бон-прокси**              | Да                 | Нет              |

### Описание компонентов

| Компонент           | Описание                                                                                                                                                            |
| ------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Merge Armature**  | Объединяет арматуру префабов с родительским аватаром, обычно для добавления одежды. MA минимизирует количество создаваемых костей, повторно используя существующие. |
| **Merge Animator**  | Объединяет под-аниматоры с родительским аватаром, полезно для различных типов аватарных фишек.                                                                      |
| **Object Toggle**   | Создает элементы меню для активации или деактивации объектов. Также может обновлять блендшейпы при тоглинге.                                                        |
| **Blendshape Sync** | Синхронизирует блендшейпы одежды или аксессуаров с базовым аватаром при изменении формы тела.                                                                       |
| **Bone Proxy**      | Позволяет добавлять уникальные пропсы, такие как оружие или спецэффекты, непосредственно прикрепленные к костям аватара.                                            |
| **Menu System**     | Полная система меню для редактирования аватара из меню VRChat.                                                                                                      |

> [!TIP]
> Modular Avatar особенно полезен, когда вы хотите распространять одежду или аксессуары в виде префабов. Пользователям просто нужно перетащить префаб на свой аватар, и MA всё сделает автоматически.

## Как установить?

### Установка через VCC (VRChat Creator Companion)

1. Добавьте репозиторий в VCC:
   - Нажмите: [Add Modular Avatar to VCC](vcc://vpm/addRepo?url=https://vpm.nadena.dev/vpm.json)
   - Или перейдите в **Settings** → **Packages** → **Add Repository**, вставьте URL `https://vpm.nadena.dev/vpm.json` и нажмите **Add**
2. Перейдите в **Manage Project** вашего проекта
3. В списке пакетов найдите **Modular Avatar** и нажмите **[+]** чтобы добавить
4. Нажмите **Open Project** в VCC

## Как использовать?

### Создание базового тогла

1. Щелкните правой кнопкой мыши на аватаре в Unity
2. Выберите **Modular Avatar → Create Toggle**
3. Будет создан новый GameObject с компонентами **Menu Item**, **Menu Installer** и **Object Toggle**
4. В компоненте **Object Toggle** нажмите кнопку **+** чтобы добавить запись
5. Перетащите объект, который хотите тоглить, в пустое поле
6. Готово! Тогл автоматически появится в меню вашего аватара

### Установка одежды

1. Перетащите префаб одежды на аватар
2. Щелкните правой кнопкой мыши на одежде и выберите **ModularAvatar → Setup Outfit**
3. MA автоматически настроит арматуру и анимации

> [!TIP]
> Официальный туториал можно найти в [документации Modular Avatar](https://modular-avatar.nadena.dev/docs/tutorials).

## Связь с другими инструментами

> [!TIP]
> Смотрите сравнительную таблицу выше, чтобы увидеть различия между Modular Avatar и VRCFury.

Modular Avatar и VRCFury — **дополняющие друг друга инструменты**. Многие современные предметы одежды поддерживают оба. Смотрите документацию предмета одежды, чтобы узнать, какой метод рекомендует создатель.

- **[VRCFury](/wiki?topic=vrcfury)**: Сосредоточен на установке анимаций и жестов.
- **NDMF (Non-Destructive Modular Framework)**: Базовая платформа, обеспечивающая неразрушающую обработку. Устанавливается автоматически с Modular Avatar.
