# Руководство: Подготовка Unity с VCC
Следуйте этим шагам ПЕРЕД импортом загруженного аватара

### Шаг 1: Установить VRChat Creator Companion (VCC)
Скачайте **VRChat Creator Companion** с [vrchat.com/home/download](https://vrchat.com/home/download). **VCC** - это официальный инструмент, который автоматически управляет Unity, VRChat SDK и всеми необходимыми пакетами.

### Шаг 2: Создать новый проект аватара
Откройте VCC → **Projects** → **Create New Project**. Выберите шаблон **"Avatars"**. Дайте ему имя (например, "Мои Аватары VRChat"). VCC автоматически установит **Unity** и **VRChat SDK**.

### Шаг 3: Добавить репозиторий Poiyomi
В VCC перейдите в **Settings** → **Packages** → **Add Repository**. Вставьте этот URL: [https://poiyomi.github.io/vpm/index.json](https://poiyomi.github.io/vpm/index.json) и нажмите "Add". Это позволит вам легко установить **Poiyomi**.

### Шаг 4: Добавить репозиторий VRCFury (Опционально)
Если вашему аватару это требуется, в **Settings** → **Packages** → **Add Repository**, вставьте: [https://vcc.vrcfury.com](https://vcc.vrcfury.com) и нажмите "Add". **VRCFury** упрощает установку одежды и аксессуаров.

### Шаг 5: Установить пакеты в ваш проект
В VCC выберите ваш проект → **Manage Project**. Найдите **"Poiyomi Toon Shader"** и нажмите кнопку **"+"**, чтобы добавить его. Если вам нужен VRCFury, добавьте его тоже. Нажмите **"Apply"**.

### Шаг 6: Открыть проект и импортировать аватар
В VCC нажмите **"Open Project"**, чтобы запустить Unity. После открытия импортируйте ваш аватар: перетащите файл **.unitypackage** в окно Unity или используйте **Assets → Import Package → Custom Package**.

### Шаг 7: Проверить и настроить
Перетащите **префаб аватара** на сцену. Если все правильно, вы **НЕ увидите материалы пурпурного цвета** (розовый). Настройте аватар с помощью **VRChat SDK → Show Control Panel → Builder**. Исправьте ошибки с помощью **"Auto Fix"** и загрузите с помощью **"Build & Publish"**.

> [!TIP]
> Важный совет
> VCC упрощает ВСЁ. Вам больше не нужно вручную устанавливать Unity или искать правильную версию. VCC делает это автоматически. Всегда используйте VCC для управления вашими проектами VRChat.
